# LIFTX Complete Site Mockup

This package is the full-site mockup, not a homepage-only patch.

## Included
- 14 HTML pages
- Shared site design system
- Homepage styling
- Brands styling
- FAQ search and accordion
- Contact form mockup
- Sitemap
- Robots file
- Cloudflare extensionless URL rewrites
- Optimized real LIFTX photography
- LIFTX wordmark and favicon

## Branch workflow
1. Create a NEW branch from the currently working production branch or last known-good production commit.
2. Suggested branch name: `full-site-cleanup-mockup`
3. Upload every file and folder in this package to the repository root.
4. Do not merge into `main`.
5. Review the Cloudflare branch preview on mobile first, then desktop.
6. Verify the `_redirects` behavior in the Cloudflare preview.
7. Replace the mockup email form with the verified production form endpoint before launch.

## Important
This build preserves the existing architecture. Full-View is positioned as the premium specialty while repair, residential, commercial, builder, and operator work remain core services.
